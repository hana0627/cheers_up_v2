# cheers_up - 카카오길찾기를 이용한 주변 술집찾기 프로젝트

카카오 길찾기를 이용한 술집찾기 프로젝트입니다.

---

## 개발 환경

* Intellij IDEA Ultimate
* Java 17
* Gradle 7.6.1
* Spring Boot 2.7.10

---

## 기술 세부 스택

Spring Boot

* Spring Web
* Spring Data JPA
* Thymeleaf
* Spring Security
* Oauth2
* Jwt
* MariaDB
* Docker-compose
* Lombok


---

## 개발기간

1차 : 2023.04.16 ~ 2023.04.23
2차 : 2025.05. ~

---

## 주요기능

* 카카오 로그인기능을 이용한 회원가입
* 카카오 주소검색을 기반으로 카카오 카테고리검색
