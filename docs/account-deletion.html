Cheers Up 계정 삭제 안내

Cheers Up은 별도의 회원가입 없이 카카오 로그인만을 사용합니다.
앱 내에 저장되는 개인정보는 없으며, 로그인 정보는 카카오에서 관리합니다.

카카오 연결 해제 방법

  1. 카카오톡 앱 실행
  2. [더보기] → [전체 서비스] → [카카오계정] 선택
  3. [연결된 서비스 관리] 메뉴로 이동
  4. Cheers Up 앱 선택 후 [연결 끊기]

이 방법을 통해 Cheers Up에서 로그인한 계정을 완전히 삭제할 수 있습니다.
