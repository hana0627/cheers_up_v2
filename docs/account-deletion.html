Cheers Up 계정 삭제 안내

앱 이름: Cheers Up
개발자: 박하나

Cheers Up은 별도의 회원가입 없이 카카오 로그인(OAuth)을 사용합니다.
따라서 사용자가 앱에서 로그인한 계정을 삭제하고자 할 경우, 카카오 계정의 연결 해제를 통해 계정 연결을 끊을 수 있습니다.

계정 삭제(카카오 연결 해제) 방법

  1. 카카오톡 앱 실행
  2. [더보기] → [전체 서비스] → [카카오계정] 선택
  3. [연결된 서비스 관리] 메뉴로 이동
  4. Cheers Up 앱 선택 후 [연결 끊기] 선택

삭제 또는 보관되는 데이터 유형 및 보관 기간

  • Cheers Up은 앱 사용 통계를 분석하고 사용자 경험을 개선하기 위해 일부 사용자 정보를 서버에 저장합니다.
  • 수집되는 정보는 기기 고유 ID(또는 사용자 고유 ID) 및 카카오톡 닉네임, 위치 기반 데이터 등으로, 사용자 식별을 최소화한 형태입니다.
  • 해당 데이터는 카카오 로그인을 통해 연결된 사용자 ID를 기준으로 저장되며, 계정 연결 해제 시 자동으로 삭제됩니다.
  • 또한, 사용자 요청 시 개발자는 모든 관련 데이터를 7일 이내에 완전 삭제합니다.

이 페이지는 사용자가 계정 연결을 해제하거나 개인정보와 관련된 요청을 할 수 있도록 제공됩니다.
